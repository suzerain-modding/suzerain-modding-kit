# Inspecting Game Data

When modding, you often need identifiers and other technical details for game assets in order to make changes. For example, the "Articy ID" of a conversation node is required when hooking into it.

[Suzerain Data Dumper](https://github.com/suzerain-modding/suzerain-data-dumper) is a mod that exports game data to JSON. [Suzerain Data Viewer](https://github.com/suzerain-modding/suzerain-data-viewer) reads that JSON and generates a static HTML site so the data can be inspected easily.

See the Suzerain Data Dumper [README.md](https://github.com/suzerain-modding/suzerain-data-dumper/blob/main/README.md) and the Suzerain Data Viewer [README.md](https://github.com/suzerain-modding/suzerain-data-viewer/blob/main/README.md) for usage instructions.
