# Uninstalling Mods

This guide has two sections:

- **Uninstall Mods:** Remove a mod or mods from Suzerain.
- **Uninstall MelonLoader:** Remove MelonLoader from Suzerain.

If you want to disable mods without uninstalling them, see [Disabling Mods](disabling-mods.md).

## Uninstall Mods

If a mod provides its own uninstallation instructions, follow them. Otherwise, follow the instructions below.

To uninstall most mods:

1. Launch Steam if it's not already open.
2. Select Suzerain in your library.
3. Select the gear icon > Manage > Browse local files.
4. Open the `Mods` folder.
5. Delete the DLLs of the mods you no longer want.
